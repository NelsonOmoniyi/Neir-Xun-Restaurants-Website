== Description ==
Rishi companion is a plugin that offers powerful features for Rishi theme. It includes features to speed your website and tune fine your website.

== Key Features ==

**_The Rishi companion includes the following features:_**

* Cookies Consent Extension: It allows you to add the GDPR notice to your website.
* Customizer Reset Extension: It allows you to quickly reset your customizer to default settings.
* Transparent Header Extension: It allows you to add a transparent header to your website.
* Performance Extension: It includes the settings to quickly disable the unwanted settings and boost your website speed.
* Social Sharing: It allows you to add a social sharing option to your single posts.
* Reading Progress Bar: Add a reading progress bar on your website to show the reading progress to your readers.

**_Rishi companion is packed with performance-boosting features such as:_**

* Smart Lazy Load for images with loading animation
* Remove Version parameters from CSS and JS files
* Remove default WP Emoji files
* Disable self-pingbacks
* Disable RSS Feeds
* Defer JavaScript files to improve the loading speed

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 7.0 or greater

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select rishi-companion.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download rishi-companion.zip
2. Extract the rishi-companion directory to your computer
3. Upload the rishi-companion directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
